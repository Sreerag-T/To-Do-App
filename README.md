# To-Do List App

A simple to-do list app built using React and Vite.

## Features
- Add new tasks
- Mark tasks as completed
- Edit tasks
- Delete tasks

## Tech Used
- React (functional components, hooks)
- Vite
- CSS

## How to Run

1. Install dependencies
npm install

2. Start the app
npm run dev

3. Open the link shown in the terminal (usually http://localhost:5173)
